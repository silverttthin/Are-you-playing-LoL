
# LoL Alarm Program
This is very very simple program that send an alarm when someone is playing league of legends.

지정한 사용자가 롤을 하면 게임개시 5분동안 슬랙 메신저로 알람을 지속적으로 보냅니다.     
관련 서비스를 웹으로 개발하고 배포할 계획을 가지고 있습니다.
